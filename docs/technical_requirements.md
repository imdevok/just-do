# 1. Цель проекта

Цель проекта - разработать систему отслеживания задач (далее Система). Пользователь может создавать в Системе списки задач, управлять ими (менять название, удалять). Также он сможет создавать задачи, выполнять их, сортировать их по статусу: `выполненные`, `активные`, `все`.

# 2. Описание системы
Система состоит из следующих основных функциональных блоков:
1. Компонент списка задач
2. Компонент pomodoro-таймер
3. Функционал напоминаний
4. Функционал сортировки задач по разным категориям

## 2.1. Список задач
Пользователь сможет создать собственный список задач, назвать по-своему его или вовсе удалить, а также создать саму задачу, поставить для неё напоминание.

## 2.2. Pomodoro-техника
После нажатия на задачу, пользователь может выбрать pomodoro-таймер для эффективного выполнения задачи. Интервал работы и отдыха будут редактируемыми.

## 2.3.  Напоминания
При создании задачи пользователь может выбрать время для напоминания ему о задаче.

## 2.4. Сортировка категорий
В списке задач можно отсортировать задачи по их статусу (`выполненные`, `активные`, `все`).

## 3. Предлагаемый стек технологий
Для реализации системы предлагается следующий стек технологий:
* Frontend:
    - SolidJS
    - TypeScript

## 4. Требования к дизайну
Минимализм, лаконичность, акцент на контент. Белый фон. В нижней части страницы (в подвале) должно быть написано:

«Работает на Open Source» со ссылкой на GitHub проекта.
